## Arsenal

### webScan

 用于内网web服务探测(80、443端口)，使用python2内置模块编写，无需其它依赖。

 探测一个C段大约2分半左右。

#### 使用



python webScan.py --host xxx.xxx.xxx.xxx/24

python webScan.py --host xxx.xxx.xxx.xxx
